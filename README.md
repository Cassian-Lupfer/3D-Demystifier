This is a small C++ project with Raylib inspired by a video of Tsoding! (Link: https://youtu.be/qjWkNZ0SXfo)
I found this way of making 3D Visuals with simple formulas very interesting. That is why I interpreted it into a C++ project to learn this programming language!
In this little software you can move the camera and spin the cube that is positioned in the origin. Have fun!
